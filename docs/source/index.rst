CamCOPS: the Cambridge Cognitive and Psychiatric Assessment Kit
===============================================================

.. image:: images/camcops_icon_500.png

CamCOPS is an open-source application for capturing information relevant for
cognitive and psychiatric assessment, on tablets, laptops, and desktops. It
offers simple questionnaires and more complex tasks, and sends its data
securely to your server.
